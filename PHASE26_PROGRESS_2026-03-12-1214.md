### 1. MP3 导出支持 (100%)

**修改文件**: `DreamMusicExportService.swift`

**新增功能**:
- ✅ MP3 格式导出支持
- ✅ 使用 AVAssetExportSession 进行格式转换
- ✅ 导出进度跟踪
- ✅ 临时文件管理
- ✅ 错误处理增强

**技术实现**:
```swift
enum ExportFormat {
    case aac      // AAC (.m4a)
    case wav      // WAV (.wav)
    case mp3      // MP3 (.mp3) ✨ 新增
}
```

**导出流程**:
1. 生成音频缓冲区
2. 创建临时 CAF 文件
3. 使用 AVAssetExportSession 转换为 MP3
4. 清理临时文件
5. 返回导出文件 URL
