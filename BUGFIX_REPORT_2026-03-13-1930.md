## 🔧 修复的问题

### 1. SpeechSynthesisService.swift - 强制解包 AVSpeechSynthesisVoice（低优先级）

**问题**: 在语音预览按钮的 fallback 逻辑中使用了 `AVSpeechSynthesisVoice(language: "zh-CN")!` 强制解包。虽然 "zh-CN" 是有效的语言标识符，但强制解包仍然是不安全的编程实践。

**文件**: `DreamLog/SpeechSynthesisService.swift`

**修复前**:
```swift
Button("🔊 预览选中语音") {
    if let voiceId = speechService.config.voiceIdentifier,
       let voice = AVSpeechSynthesisVoice(identifier: voiceId) {
        speechService.previewVoice(voice)
    } else {
        speechService.previewVoice(AVSpeechSynthesisVoice(language: "zh-CN")!)
    }
}
```

**修复后**:
```swift
Button("🔊 预览选中语音") {
    if let voiceId = speechService.config.voiceIdentifier,
       let voice = AVSpeechSynthesisVoice(identifier: voiceId) {
        speechService.previewVoice(voice)
    } else if let defaultVoice = AVSpeechSynthesisVoice(language: "zh-CN") {
        speechService.previewVoice(defaultVoice)
    }
}
```

**影响**: 
- 消除了强制解包，使用可选绑定
- 当默认语音不可用时，按钮不执行任何操作（而不是崩溃）
- 更符合 Swift 安全编程最佳实践
