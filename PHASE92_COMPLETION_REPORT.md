## 🔧 技术实现要点

### 生物识别

- 使用 LocalAuthentication 框架
- 支持 Face ID 和 Touch ID
- Fallback 到设备密码
- 处理生物识别不可用情况
- 可配置锁定超时

### 加密

- 使用 CryptoKit 进行 AES-256-GCM 加密
- 密钥存储在 Keychain (安全区域)
- 每次加密使用随机 nonce (IV)
- 支持标题和内容分别加密

### 数据保护

- 使用 SwiftData 进行数据持久化
- 软删除机制 (回收站)
- 30 天自动清理
- 备份文件压缩 (LZ4)

### UI/UX

- SwiftUI 原生界面
- 流畅的动画过渡
- 清晰的状态反馈
- 符合 Human Interface Guidelines
