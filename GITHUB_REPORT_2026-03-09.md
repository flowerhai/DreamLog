### Phase 13: AI 梦境助手增强

#### 1. 语音对话模式 🎙️

**功能描述**: 完整的语音交互体验，让 AI 助手能够"说话"和"听话"

**技术实现**:
- TTS (Text-to-Speech): 使用 SpeechSynthesisService 朗读助手回复

**用户体验提升**:
- 无需阅读即可获取助手回复
- 语音输入更自然快捷
- 适合睡前/驾驶等场景

**代码变更**:

## 🐛 Bug 修复

### ViewImageRenderer 递归调用问题

**严重程度**: 🔴 Critical

**问题描述**: 
UIImage 扩展中的 `pngData()` 和 `jpegData(compressionQuality:)` 方法调用自身，导致无限递归和栈溢出。

**影响范围**: 
- 分享卡片图片导出功能 (Phase 11.5)
- 用户尝试导出梦境回顾分享卡片时应用崩溃

**修复方案**:
```swift
// 修复前 ❌
extension UIImage {
    func pngData() -> Data? {
        return self.pngData()  // 递归调用自身!
    }
}

// 修复后 ✅
extension UIImage {
    func toPngData() -> Data? {
        return self.pngData()  // 调用原始系统方法
    }
}
```

**验证**:
- ✅ 递归调用已消除
- ✅ 方法命名避免冲突
- ✅ 调用站点已更新
- ✅ 功能测试通过
