## 📝 技术说明

### 为什么 @MainActor 很重要？

在 SwiftUI 中，ObservableObject 类通常包含 @Published 属性，这些属性的变化会触发 UI 更新。标记 @MainActor 确保：

1. **线程安全**: 所有属性访问和修改都在主线程执行
2. **UI 一致性**: 避免跨线程 UI 更新导致的未定义行为
3. **编译器检查**: Swift 编译器会检查并发安全性
4. **最佳实践**: 符合 Apple 的 SwiftUI 开发指南

### 修复影响

- **无破坏性变更**: @MainActor 是添加性标记，不影响现有 API
- **性能无影响**: 主线程执行是 SwiftUI 的预期行为
- **提高稳定性**: 避免潜在的竞态条件和 UI 更新问题
