## 截图要求

### Apple 官方要求

**分辨率要求**:

| 设备类型 | 分辨率 (像素) | 比例 |
|---------|-------------|------|
| iPhone 15 Pro Max | 1290 x 2796 | 19.5:9 |
| iPhone 15 Pro | 1179 x 2556 | 19.5:9 |
| iPhone 14 Pro Max | 1290 x 2796 | 19.5:9 |
| iPhone 14 Pro | 1179 x 2556 | 19.5:9 |
| iPhone 8 Plus / 7 Plus | 1242 x 2208 | 16:9 |

**格式要求**:
- 文件格式：PNG 或 JPEG
- 颜色模式：RGB
- 最大文件大小：10MB

**内容要求**:
- ✅ 展示应用实际界面
- ✅ 突出核心功能
- ✅ 清晰易读
- ❌ 不要包含设备边框 (除非使用 mockup)
- ❌ 不要包含敏感信息
- ❌ 不要包含占位符文本

## 使用截图辅助视图

### 步骤 1: 打开 Xcode Preview

1. 打开 `DreamLogScreenshotHelper.swift`
2. 在右侧边栏选择 Preview
3. 选择要预览的视图

```swift
#Preview("首页截图") {
    NavigationStack {
        HomeScreenshotView()
    }
}
```

### 步骤 2: 调整设备

1. 在 Preview 顶部选择设备型号
2. 推荐使用：
   - iPhone 15 Pro Max (最大尺寸)
   - iPhone 15 Pro (主流尺寸)

### 步骤 3: 截取屏幕

**方法 A: Xcode 截图**
1. 在 Preview 中右键
2. 选择 "Copy Image"
3. 粘贴到图像编辑软件

**方法 B: Simulator 截图**
1. 在 Simulator 中运行应用
2. 导航到对应页面
3. 使用快捷键：
   - macOS: `Cmd + S`
   - 或 File → Save Screen

**方法 C: 真机截图**
1. 在真机上运行应用
2. 导航到对应页面
3. 使用物理按键截图

## 截图步骤

### 准备工作

1. **清理数据**
   - 确保示例数据美观
   - 移除敏感或测试数据
   - 使用有意义的示例文本

2. **设置主题**
   - 使用浅色模式 (推荐)
   - 或暗色模式 (展示多样性)
   - 确保文字清晰可读

3. **准备设备**
   - 使用最新 iPhone 型号
   - 确保电量充足
   - 关闭通知 (避免干扰)

### 执行截图

#### 截图 1: 首页

```
1. 打开应用 → 首页
2. 确保显示统计卡片
3. 确保梦境列表有 3-5 条记录
4. 截图保存为：home_6.7.png
```

#### 截图 2: AI 解析

```
1. 选择一个梦境
2. 点击 "AI 解析" 标签
3. 等待解析完成
4. 确保三层解析都显示
5. 截图保存为：ai_analysis_6.7.png
```

#### 截图 3: AR 可视化

```
1. 选择一个梦境
2. 点击 "AR 可视化" 标签
3. 等待 AR 场景加载
4. 确保 3D 元素显示
5. 截图保存为：ar_view_6.7.png
```

#### 截图 4: 分享中心

```
1. 导航到 "分享中心"
2. 确保显示统计和平台按钮
3. 确保有分享配置
4. 截图保存为：share_hub_6.7.png
```

#### 截图 5: 音乐/视频

```
1. 导航到 "梦境音乐" 或 "梦境视频"
2. 选择一个生成的音乐/视频
3. 显示播放界面
4. 截图保存为：music_video_6.7.png
```

## 后期处理

### 推荐工具

- **Figma** (免费，在线)
- **Sketch** (macOS)
- **Photoshop** (专业)
- **Canva** (在线，模板丰富)
- **Xcode** (内置，简单)

### 处理步骤

#### 1. 添加设备框架 (可选)

```
1. 导入截图到设计工具
2. 添加 iPhone 设备框架
3. 调整截图适配框架
4. 导出带框架的版本
```

**资源**:
- [Apple Design Resources](https://developer.apple.com/design/resources/)
- [Mockup World](https://www.mockupworld.co/)

#### 2. 添加文案说明

```
1. 在截图上方或下方添加文字
2. 使用简洁有力的文案
3. 字体：SF Pro 或类似无衬线字体
4. 字号：标题 24-32pt，副标题 16-20pt
5. 颜色：与截图主色调协调
```

**文案模板**:
```
主标题 (24-32pt, 粗体)
副标题 (16-20pt, 常规)
```

#### 3. 调整颜色和对比度

```
1. 确保文字清晰可读
2. 调整亮度/对比度
3. 统一所有截图的色调
4. 检查是否有过曝或过暗
```

#### 4. 导出最终版本

```
格式：PNG
分辨率：原始分辨率 (不要缩放)
文件大小：<10MB
命名：功能_设备尺寸.png
```

## 上传到 App Store

### 步骤 1: 登录 App Store Connect

```
1. 访问 https://appstoreconnect.apple.com
2. 登录开发者账号
3. 选择 "我的 App"
4. 选择 DreamLog 应用
```

### 步骤 2: 进入版本页面

```
1. 左侧菜单选择 "App Store" 标签
2. 选择要编辑的版本 (如 1.0)
3. 滚动到 "版本信息" 部分
```

### 步骤 3: 上传截图

```
1. 找到 "截图" 部分
2. 选择设备类型 (如 "6.7 英寸 (iPhone 15 Pro Max)")
3. 拖拽截图到上传区域
4. 调整截图顺序 (从上到下显示)
5. 重复其他设备尺寸
```

### 步骤 4: 添加截图说明 (可选)

```
1. 点击截图下方的 "添加说明"
2. 输入简短描述 (可选)
3. 帮助审核人员理解功能
```

### 步骤 5: 保存并提交

```
1. 点击右上角 "保存"
2. 检查所有截图显示正确
3. 继续填写其他元数据
4. 提交审核
```

## 常见问题

### Q: 需要多少种设备尺寸的截图？

**A**: 至少需要一种尺寸 (6.7 英寸或 6.5 英寸)。建议提供:
- 6.7 英寸 (iPhone 15 Pro Max / 14 Pro Max)
- 6.1 英寸 (iPhone 15 Pro / 14 Pro)

### Q: 可以使用设备框架吗？

**A**: 可以，但不是必须的。Apple 会在 App Store 中自动添加设备框架。如果你添加了自己的框架，确保:
- 框架美观专业
- 不影响界面内容展示
- 所有截图使用相同框架

### Q: 截图需要本地化吗？

**A**: 如果应用支持多语言，建议为每个语言创建对应的截图。至少应该:
- 中文截图 (主要市场)
- 英文截图 (国际市场)

### Q: 截图更新频率？

**A**: 建议:
- 每次大版本更新时更新截图
- 添加新功能时补充截图
- 根据用户反馈优化截图
