- ✅ 创建 `createCollection(from:)` 方法
- ✅ 支持收藏集元数据追踪

**同步逻辑**:
```swift
// 上传本地收藏集
for collection in localCollections {
    let record = CKRecord(recordType: "collection", ...)
    record["name"] = collection.name
    record["color"] = collection.color
    try await database.save(record)
}

// 下载远程收藏集
for record in remoteCollections {
    if !exists {
        try createCollection(from: record)
    }
}
```

#### 3.2 梦境实体创建

- ✅ 自动创建同步元数据

#### 3.3 冲突处理
- ✅ 实现 `handleConflict(for:localMetadata:)` 方法
- ✅ 支持 4 种冲突解决策略:
  - `.keepLocal`: 保留本地版本
  - `.keepRemote`: 保留远程版本
  - `.keepNewest`: 保留最新版本
  - `.manual`: 标记为手动解决
- ✅ 根据配置自动处理冲突

#### 3.4 冲突解决
- ✅ 实现 `resolveConflict(_:choice:)` 方法
- ✅ 支持 3 种解决选项:
  - `.keepLocal`: 保留本地并更新远程
  - `.keepRemote`: 应用远程版本到本地
  - `.merge`: 合并两个版本内容
- ✅ 合并时保留双方内容并标记

#### 3.5 同步重置
- ✅ 实现 `resetSyncState()` 方法
