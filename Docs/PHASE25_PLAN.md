# Phase 25 - App Store 发布准备与用户体验优化 🚀✨

**Phase 25 目标**: 为 App Store 发布做全面准备，优化用户体验，添加引导流程

---

## 📋 功能模块

### 25.1 应用内购买与订阅系统 💎

- [ ] **订阅层级设计**
  - 免费版：基础梦境记录、AI 解析（每日 3 次）、基础统计
  - 高级版（月费/年费）：无限 AI 解析、高级统计、AR 功能、视频导出、云存储
  - 终身版：一次性购买，解锁所有功能

- [ ] **IAP 集成**
  - StoreKit 2 集成
  - 订阅状态管理
  - 恢复购买功能
  - 订阅设置界面

- [ ] **付费墙设计**
  - 精美的功能对比卡片
  - 免费试用引导
  - 价格展示（本地化）

---

### 25.2 新用户引导流程 🎓

- [ ] **Onboarding 页面** (4-5 页)
  - 欢迎页面：DreamLog 介绍
  - 功能亮点：语音记录、AI 解析、梦境画廊
  - 隐私承诺：本地优先、端到端加密
  - 权限请求：麦克风、通知、健康数据（可选）

- [ ] **首次使用引导**
  - 快速教程（可跳过）
  - 示例梦境展示
  - 目标设置引导

- [ ] **权限说明**
  - 麦克风：用于语音记录梦境
  - 通知：用于智能提醒
  - 健康 App：用于睡眠分析（可选）
  - 照片：用于保存梦境图片

---

### 25.3 App Store 资源准备 📱

- [ ] **应用截图** (6.5 寸和 5.5 寸)
  - 首页：快速记录界面
  - AI 解析：梦境分析结果
  - 数据统计：图表和洞察
  - 梦境画廊：AI 绘画展示
  - AR 体验：3D 梦境世界
  - 冥想功能：音效和引导

- [ ] **应用预览视频** (30 秒)
  - 开场：DreamLog Logo
  - 核心功能演示
  - 用户评价展示
  - 结尾：下载引导

- [ ] **应用元数据**
  - 应用名称：DreamLog - AI 梦境日记
  - 副标题：记录梦境，发现潜意识
  - 关键词：梦境，日记，AI，冥想，睡眠
  - 描述：完整功能介绍（中英文）
  - 隐私政策 URL
  - 支持 URL
