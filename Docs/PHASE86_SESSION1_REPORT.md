### 🎯 核心功能

#### 1. 氛围音景系统

**6 种音景分类**:
- 🌿 自然类 - 雨声/海浪/森林/溪流
- 🏙️ 城市类 - 交通/咖啡馆/城市环境
- 🏠 室内类 - 壁炉/时钟/翻书声
- ✨ 奇幻类 - 魔法/太空/梦境
- 〰️ 白噪音 - 白/粉红/棕色噪音
- 🧘 冥想类 - 双耳节拍/颂钵

**14+ 种预设音景**:
- ⛈️ 暴风雨夜 - 大雨 + 远雷 + 风声
- 🌲 森林清晨 - 鸟鸣 + 树叶 + 溪流
- 🌊 海边日落 - 海浪 + 海鸥 + 微风
- 🌧️ 绵绵细雨 - 细雨 + 屋檐滴水
- 💧 山间溪流 - 溪流 + 水击石
- 🔥 温暖壁炉 - 壁炉 + 时钟滴答
- 🕰️ 时钟滴答 - 落地钟
- 📖 翻书声 - 翻书 + 图书馆环境
- 〰️ 白噪音 - 均匀白噪音
- 📊 粉红噪音 - 柔和低频噪音
- 📈 棕色噪音 - 深沉低频噪音
- 🧘 θ波双耳节拍 - 深度放松
- 🧠 α波双耳节拍 - 放松创意
- 🔔 颂钵冥想 - 传统颂钵

#### 2. 多层音景混合

- 支持多层音景叠加播放
- 每层独立音量控制 (0.0-1.0)
- 声像控制 (左 - 右)
- 渐入渐出效果
- 循环播放支持
- 音调调节 (0.5-2.0)

#### 3. 睡眠定时器

**5 种时长预设**:
- 15 分钟
- 30 分钟
- 1 小时
- 1.5 小时
- 2 小时

**3 种结束动作**:
- 停止播放
- 降低音量
- 切换到白噪音

**渐出效果**:
- 可配置渐出时长 (默认 30 秒)
- 平滑音量过渡

#### 4. 智能推荐算法

**基于情绪推荐**:
- 平静 → 自然/冥想/白噪音
- 快乐 → 自然/奇幻
- 焦虑 → 白噪音/冥想/自然
- 悲伤 → 自然/室内
- 困惑 → 冥想/白噪音
- 恐惧 → 白噪音/室内
- 兴奋 → 奇幻/城市

**基于内容推荐**:
- 水/海/河/雨 → 海洋/雨声音景
- 森林/树/鸟 → 森林音景
- 火/温暖 → 壁炉音景
- 高清晰度 (≥4) → α波双耳节拍

## 🎯 使用场景

### 睡前放松
- 选择"暴风雨夜"或"海边日落"音景
- 设置 30 分钟睡眠定时器
- 自动渐出停止，帮助入睡

### 冥想练习
- 选择"θ波双耳节拍"或"颂钵冥想"
- 配合梦境冥想功能使用
- 帮助进入深度放松状态

### 专注工作
- 选择"白噪音"或"图书馆环境"
- 遮蔽外界干扰
- 提高注意力集中度

### 梦境回忆
- 播放与梦境情绪匹配的音景
- 帮助进入相似情绪状态
- 促进梦境细节回忆
