## 集成点

### 与现有功能的集成
1. **梦境记录**: 记录时自动更新 `recordDreams` / `recordWithEmotions` / `recordWithTags` / `recordWithAudio` 挑战
2. **清醒梦**: 标记清醒梦时自动更新 `lucidDream` 挑战
3. **分享功能**: 分享梦境时自动更新 `shareDream` 挑战
4. **冥想功能**: 完成冥想时自动更新 `meditation` 挑战
5. **AI 解析**: 使用 AI 解析时自动更新 `analyzeDream` 挑战
6. **梦境孵育**: 完成孵育时自动更新 `incubation` 挑战

### 建议的集成代码

```swift
// 在 DreamRecordService 中
func saveDream(_ dream: Dream) async throws {
    // ... 保存逻辑
    
    // 自动更新挑战进度
    try challengeService.onDreamRecorded(
        dreamId: dream.id,
        hasEmotions: !dream.emotions.isEmpty,
        hasTags: !dream.tags.isEmpty,
        hasAudio: dream.audioURL != nil,
        isLucid: dream.isLucid
    )
}

// 在 DreamShareService 中
func shareDream(_ dream: Dream) async throws {
    // ... 分享逻辑
    
    // 自动更新挑战进度
    try challengeService.onDreamShared(dreamId: dream.id)
}

// 在 DreamMeditationService 中
func completeMeditation() async throws {
    // ... 完成逻辑
    
    // 自动更新挑战进度
    try challengeService.onMeditationCompleted()
}
```
