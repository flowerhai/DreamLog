# DreamLog 预览视频制作脚本 🎬

## 📋 视频概述

**应用名称**: DreamLog - AI 梦境日记  
**目标平台**: App Store (iOS)  
**视频用途**: App Store 产品页面预览视频  
**目标受众**: 对梦境记录、心理健康、自我探索感兴趣的用户

**核心信息**:
- 🌙 30 秒快速记录梦境
- 🧠 AI 深度解析梦境含义
- 📊 智能洞察发现模式
- 🥽 AR 技术让梦境可视化

## 🎬 分镜脚本

### 镜头 1: 开场动画 (0-3 秒)

**画面**:
- 黑色背景渐变到深蓝色星空
- DreamLog 应用图标从中心放大淡入
- 图标下方文字淡入："DreamLog - AI 梦境日记"

**动画**:
- 0-1s: 背景渐变 (黑 → 深蓝星空)
- 1-2s: 图标缩放 (50% → 100%) + 发光效果
- 2-3s: 文字从下向上淡入

**音效**:
- 轻柔的风铃声 (开场)
- 轻微的环境音 (星空氛围)

**文案** (屏幕文字):
```
DreamLog
AI 梦境日记
```

**旁白** (可选):
> "DreamLog - 你的 AI 梦境日记"

---

**动画**:
- 3-4s: 首页界面滑入

- 6-7s: 松开按钮，保存成功提示
- 7-8s: 新梦境卡片添加到列表

**音效**:
- 轻柔的点击音 (按住按钮)
- 持续的录音提示音 (低频)
- 成功的"叮"声 (保存完成)

**文案** (屏幕文字):
```
按住说话

```

**旁白**:
> "按住说话，30 秒完成梦境记录"

**技术要点**:
- 展示语音转文字的实时效果
- 突出录音界面的简洁设计
- 显示保存后的成功反馈

## 🎵 音频设计

### 背景音乐

**风格**: 轻柔、梦幻、空灵  
**乐器**: 钢琴 + 合成器 + 环境音  
**节奏**: 60-70 BPM (缓慢放松)  
**音量**: -20dB (不盖过旁白)

**推荐曲目**:
- "Dreamy Ambient Piano" (免版税)
- "Celestial Meditation" (免版税)
- 自定义原创音乐 (推荐)

| 时间点 | 音效 | 描述 | 音量 |
|--------|------|------|------|
| 0s | 风铃声 | 开场氛围 | -25dB |

| 6s | 成功音 | 保存完成 | -15dB |

## 🛠️ 制作流程

### 前期准备 (1 小时)

1. **环境设置**
   - 准备 iPhone 15 Pro Max 真机或模拟器
   - 清空通知中心
   - 准备示例梦境数据 (10+ 条)
   - 测试所有功能正常运行

2. **脚本确认**
   - 确认分镜脚本
   - 确认旁白文案
   - 确认音乐和音效
   - 确认品牌色彩和字体

### 录制阶段 (2 小时)

1. **屏幕录制**
   - 使用 macOS 内置录屏 (Cmd+Shift+5)
   - 或使用 QuickTime Player
   - 分辨率：1920x1080
   - 帧率：60fps

2. **镜头拍摄**
   - 按照分镜脚本依次拍摄
   - 每个镜头拍摄 3-5 条备选
   - 确保操作流畅无卡顿
   - 注意手势和动画的完整性

3. **旁白录制**
   - 在安静环境中录制
   - 使用专业麦克风
   - 按照脚本分段录制
   - 保留原始 WAV 文件

### 后期制作 (2-3 小时)

1. **素材整理**
   - 筛选最佳镜头
   - 按时间线排列
   - 标记需要调整的片段

2. **剪辑合成**
   - 使用 Final Cut Pro / Premiere Pro / DaVinci Resolve
   - 按照分镜时间线剪辑
   - 添加转场效果
   - 调整色彩和亮度

3. **音频处理**
   - 添加背景音乐
   - 添加音效
   - 混音旁白和音乐
   - 音量标准化 (-1dB 峰值)

4. **字幕添加**
   - 添加屏幕文字
   - 添加字幕 (可选)
   - 检查拼写和排版
   - 确保文字清晰可读

5. **导出优化**
   - 格式：MP4 (H.264)
   - 分辨率：1920x1080
   - 帧率：30fps
   - 码率：10-15 Mbps
   - 音频：AAC, 44.1kHz, 128kbps
   - 文件大小：< 100MB

### 质量检查 (30 分钟)

- [ ] 视频流畅无卡顿
- [ ] 音频清晰无杂音
- [ ] 字幕无错别字
- [ ] 时长符合要求 (25-30 秒)
- [ ] 文件大小符合要求 (< 100MB)
- [ ] 在不同设备上预览效果

## 📤 App Store Connect 上传

### 上传步骤

1. 登录 [App Store Connect](https://appstoreconnect.apple.com)
2. 选择 "我的 App" → DreamLog
3. 选择版本 (1.0.0)
4. 找到 "App 预览和截图" 部分
5. 选择设备类型 (iPhone 6.7 英寸)
6. 点击 "+" 上传预览视频
7. 选择制作好的 MP4 文件
8. 等待上传和转码完成
9. 预览效果确认
10. 保存

### 注意事项

- 视频会自动转码为 Apple 要求的格式
- 上传后需要几分钟处理时间
- 可以在多个设备类型上传不同视频
- 视频会 autoplay (无声) 在 App Store 页面

## 💡 优化建议

### Do ✅

- 保持节奏紧凑 (每镜头 3-5 秒)
- 突出核心价值 (AI 解析/AR 可视化)
- 使用真实应用界面
- 确保文字清晰可读
- 测试在不同设备上的效果

### Don't ❌

- 避免过长的镜头 (>6 秒)
- 避免过多文字说明
- 避免使用模糊截图
- 避免嘈杂的背景音乐
- 避免展示未实现的功能
