## 📋 功能概述

AI 教练功能提供个性化的数字健康计划、习惯养成追踪和 AI 驱动的干预建议，帮助用户建立健康的生活习惯，改善睡眠质量，提升梦境回忆能力。

### 核心价值

- 🎯 **个性化计划** - 7 种预设计划模板，针对不同目标
- 📈 **习惯追踪** - 科学的习惯养成系统，连续天数统计
- 🤖 **AI 干预** - 智能检测和个性化建议
- 📊 **进度可视化** - 完整的统计和图表展示

## ✨ 核心功能

### 🎯 7 种预设计划模板

1. **7 天睡眠改善** - 通过科学方法改善睡眠质量

3. **30 天清醒梦入门** - 学习清醒梦技巧
4. **21 天压力缓解** - 管理和缓解压力

6. **28 天情绪平衡** - 建立情绪平衡
7. **30 天正念修行** - 培养正念觉察

- **创建计划** - 从预设模板快速创建
- **进度追踪** - 实时显示完成进度
- **状态管理** - 草稿/进行中/已暂停/已完成/已归档
- **连续天数** - 追踪连续完成天数

### ✅ 习惯追踪

- **12 种习惯类型**:
  - 记录梦境
  - 规律作息
  - 冥想练习
  - 现实检查
  - 梦境孵化
  - 晨间反思
  - 感恩日记
  - 屏幕时间限制
  - 咖啡因限制
  - 运动
  - 呼吸练习

- **频率设置**: 每日/工作日/周末/每周/自定义
- **提醒功能**: 可配置提醒时间
- **完成记录**: 记录每次完成的详情（心情、难度等）
- **连续统计**: 当前连续天数和最长连续天数

### 🤖 AI 智能干预

- **8 种干预类型**:

- **习惯统计**: 总习惯数/活跃习惯/完成率
- **连续追踪**: 当前连续天数/最长连续天数

## 📦 数据模型

### 核心模型

```swift
DreamAICoachPlan          // AI 教练计划
├── id: UUID
├── planType: CoachPlanType
├── status: CoachPlanStatus
├── goals: [CoachGoal]
├── habits: [DreamAICoachHabit]
├── progress: Double
└── streak: Int

DreamAICoachHabit         // 习惯
├── habitType: HabitType
├── frequency: HabitFrequency
├── completionHistory: [HabitCompletion]
├── streak: Int
└── totalCompletions: Int

DreamAICoachIntervention  // AI 干预
├── interventionType: InterventionType
├── priority: InterventionPriority
├── status: InterventionStatus
└── suggestedAction: String?
```

### 枚举类型

- `CoachPlanType` - 7 种计划类型
- `CoachPlanStatus` - 5 种计划状态
- `HabitType` - 12 种习惯类型
- `HabitFrequency` - 5 种频率选项
- `InterventionType` - 8 种干预类型
- `InterventionPriority` - 4 级优先级
- `GoalMetric` - 9 种目标指标
