## 📋 功能概述

家庭共享功能允许家庭成员之间安全地分享梦境体验，建立家庭梦境数据库，发现家族共同的梦境模式和象征。

### 核心价值

- 👨‍👩‍👧‍👦 **家庭连接** - 与家人分享梦境体验，增进理解
- 🔒 **隐私优先** - 精细的隐私控制，选择性地分享
- 🧬 **家族模式** - 发现跨代际的梦境模式和遗传特征
- 🌟 **共同体验** - 创建家庭梦境挑战和活动

---

## ✨ 核心功能

### 🏠 家庭组管理

- **创建家庭组** - 用户可创建或加入家庭组
- **邀请成员** - 通过邀请码或链接邀请家人
- **成员角色** - 管理员/成人/儿童（不同权限）
- **家庭资料** - 家庭名称、头像、成员关系

### 🔐 隐私与权限

- **选择性分享** - 用户可选择哪些梦境对家人可见
- **隐私级别**:
  - 🔒 私密 - 仅自己可见
  - 👨‍👩‍👧‍👦 家庭可见 - 家庭成员可见
  - 🌍 公开 - 所有人可见（如果启用公开功能）

- **内容过滤** - 可设置敏感内容自动对儿童隐藏
- **年龄分级** - 根据成员年龄自动调整可见内容

### 📊 家庭梦境统计

- **家庭梦境总数** - 累计梦境记录
- **活跃成员** - 本周/月活跃的家庭成员
- **共同标签** - 家庭成员共同出现的梦境元素
- **家庭梦境热力图** - 展示家庭梦境活动时间分布

### 🧬 家族梦境模式

- **遗传象征** - 跨代际重复出现的梦境符号
- **共同主题** - 家庭成员共同经历的梦境主题
- **情绪传承** - 家庭情绪模式的代际传递
- **家族梦境故事** - 自动生成家庭梦境历史

### 👨‍👩‍👧‍👦 家庭梦境活动

- **家庭梦境挑战** - 每周/月家庭梦境主题挑战
- **梦境接龙** - 家庭成员接力创作梦境故事
- **共同孵化** - 家庭一起设定梦境意图
- **梦境讨论** - 家庭群聊讨论有趣的梦境

### 🎁 家庭成就系统

- **家庭里程碑** - 家庭共同达成的成就
- **成员贡献** - 追踪每个成员的贡献
- **家庭等级** - 基于活跃度提升家庭等级
- **成就徽章** - 独特的家庭徽章收藏

## 🔧 技术实现

### 服务层

**DreamFamilyService** (Actor 并发安全)
- `createFamilyGroup()` - 创建家庭组
- `joinFamilyGroup()` - 加入家庭组
- `inviteMember()` - 邀请成员
- `removeMember()` - 移除成员
- `shareDreamWithFamily()` - 分享梦境到家庭
- `getFamilyDreams()` - 获取家庭共享梦境
- `analyzeFamilyPatterns()` - 分析家族模式
- `createFamilyChallenge()` - 创建家庭挑战
- `getFamilyStatistics()` - 获取家庭统计

**DreamFamilyPrivacyService**
- `setDreamPrivacy()` - 设置梦境隐私级别
- `getVisibleDreams()` - 获取对某成员可见的梦境
- `filterSensitiveContent()` - 过滤敏感内容
- `checkAgeAppropriate()` - 检查年龄适宜性

### 核心算法

1. **家族模式识别** - 跨成员梦境相似度分析
2. **隐私权限检查** - 多层次权限验证
3. **内容安全过滤** - 基于关键词和 AI 的内容审核
4. **家庭统计聚合** - 多成员数据汇总

## 🔒 隐私与安全

### 数据保护

- **端到端加密** - 家庭共享数据加密传输
- **本地优先** - 敏感数据优先本地存储
- **最小化共享** - 仅共享必要信息
- **可撤销访问** - 随时移除成员访问权限

### 儿童保护

- **年龄验证** - 儿童账户特殊保护
- **内容过滤** - 自动过滤不适合内容
- **家长控制** - 父母可监控和管理
- **隐私默认** - 儿童账户默认最高隐私

### 合规性

- **GDPR 合规** - 欧盟数据保护
- **COPPA 合规** - 美国儿童隐私保护
- **数据可携带** - 支持数据导出
- **删除权** - 支持完全删除账户和数据
