## 📋 开发概述

Phase 85 为 DreamLog 添加了智能建议与个性化推荐系统，基于用户的梦境数据、睡眠质量和心理状态，自动生成个性化的改善建议。

### 核心目标

- 提供 15 种智能建议类型，覆盖梦境改善、睡眠健康、心理健康、创意灵感和分析洞察
- 实现 3 级优先级系统，帮助用户识别最重要的建议
- 基于梦境模式自动生成个性化建议
- 提供精美的 UI 界面和完整的建议管理功能

## ✨ 新增功能

### 1. 15 种智能建议类型

#### 梦境改善类
- **dreamImprovement** - 梦境质量改善
- **lucidDreaming** - 清醒梦技巧
- **dreamRecall** - 梦境回忆增强

#### 睡眠健康类
- **sleepQuality** - 睡眠质量提升
- **sleepSchedule** - 作息时间建议
- **relaxationTechnique** - 放松技巧

#### 心理健康类
- **stressManagement** - 压力管理
- **moodImprovement** - 情绪改善
- **mindfulness** - 正念练习

#### 创意灵感类
- **creativeInspiration** - 创意灵感
- **writingPrompt** - 写作提示
- **artisticExpression** - 艺术表达

#### 分析洞察类
- **patternInsight** - 模式洞察
- **symbolExploration** - 符号探索
- **themeAnalysis** - 主题分析

### 2. 3 级优先级系统

| 优先级 | 标识颜色 | 说明 |
|--------|----------|------|
| 低 (low) | 灰色 | 可选阅读 |
| 中 (medium) | 蓝色 | 推荐尝试 |
| 高 (high) | 橙色 | 强烈建议 |

### 3. 智能建议生成引擎

基于以下数据自动生成建议：
- 梦境模式识别（模糊梦境、回忆困难等）
- 睡眠质量分析（0-1 评分）
- 压力水平检测（0-1 评分）
- 清醒梦频率统计
- 梦境回忆率评估
- 常见符号和主题识别

### 4. 建议模板系统

标准化模板库支持：
- 变量替换生成个性化内容
- 可执行步骤自动生成
- 预期效果说明
- 时间投入估算
- 难度等级评估（1-5 级）

### 5. 精美 UI 界面

#### 主界面 (`DreamSmartSuggestionsView`)
- 统计概览卡片（总数/进行中/已完成/完成率）
- 多维度筛选（全部/进行中/已完成/已关闭）
- 按类型筛选（15 种建议类型）
- 搜索功能

#### 建议卡片 (`SuggestionCard`)
- 类型图标和名称
- 优先级徽章
- 描述预览
- 可执行步骤预览
- 时间和难度信息
- 完成/关闭操作

#### 详情界面 (`SuggestionDetailView`)
- 完整描述
- 所有可执行步骤
- 预期效果说明
- 基于的模式展示
- 有用性评分（1-5 星）
- 完成/关闭操作

#### 配置界面 (`SuggestionConfigView`)
- 建议类型启用/禁用
- 最小优先级筛选
- 每日建议数量限制
- 通知设置
- 自动生成开关

### 6. 建议管理功能

- ✅ 标记完成/撤销完成
- ❌ 关闭建议
- ⭐ 有用性评分（1-5 星）
- 👁️ 查看次数追踪
- 📅 完成时间记录
- ⏰ 过期时间管理

### 7. 统计面板

- 总建议数 / 活跃数 / 完成数 / 关闭数
- 平均有用性评分
- 完成率计算
- 按类型分布统计
- 按优先级分布统计
- 最有帮助的类型识别
