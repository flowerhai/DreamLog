### 语音识别流程

1. **权限检查** → `SFSpeechRecognizer.requestAuthorization`
2. **创建请求** → `SFSpeechAudioBufferRecognitionRequest`
3. **创建任务** → `speechRecognizer.recognitionTask`
4. **配置音频** → `AVAudioEngine` + 输入节点 tap
5. **开始监听** → `audioEngine.start()`
6. **实时识别** → `recognitionTask` 回调
7. **命令匹配** → 遍历 `commandTriggers` 检查触发词
8. **执行命令** → 调用对应的 handler
9. **保存历史** → SwiftData 插入记录
10. **反馈** → 触觉 + 语音 (可选)

### 触发词匹配算法

```swift
func matches(_ text: String) -> Bool {
    let lowercasedText = text.lowercased()
    return triggers.contains { trigger in
        lowercasedText.contains(trigger.lowercased())
    }
}
```

简单但有效：大小写不敏感的子串匹配。

### 关键词提取

```swift
private func extractKeywords(from text: String) -> [String] {
    let stopWords = ["搜索", "查找", "找一下", "查询", "关于", "的", "我", "想", "要"]
    let words = text.components(separatedBy: CharacterSet.whitespacesAndNewlines)
    return words.filter { $0.count > 1 && !stopWords.contains($0) }
}
```

简单的停用词过滤，未来可升级为 NLP。
