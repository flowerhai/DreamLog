## 🎨 设计系统

### 配色方案

```swift
// 主色调 - 夜空紫
primary: Color("MidnightPurple")    // #6B4E9A
secondary: Color("DreamBlue")       // #4A90D9
accent: Color("StarGold")           // #FFD700

// 背景色
background: Color("DeepSpace")      // #1A1A2E
cardBackground: Color("Nebula")     // #2D2D44

// 功能色
dream: Color("DreamPurple")         // #9B7EBD
memory: Color("MemoryBlue")         // #6BB6FF
insight: Color("InsightGold")       // #FFC857
```

### 渐变色

```swift
// 夜空渐变
nightSky: LinearGradient(
    colors: [#1A1A2E, #16213E, #0F3460],
    startPoint: .top,
    endPoint: .bottom
)

// 梦境渐变
dream: LinearGradient(
    colors: [#6B4E9A, #9B7EBD, #C9B1DD],
    startPoint: .topLeading,
    endPoint: .bottomTrailing
)

// 星空渐变
starry: RadialGradient(
    colors: [#FFD700, #FFA500, .clear],
    center: .center,
    startRadius: 0,
    endRadius: 100
)
```

### 字体系统

```swift
// 神秘风格字体
largeTitle: .system(size: 34, weight: .light)
title: .system(size: 24, weight: .regular)
headline: .system(size: 18, weight: .medium)
body: .system(size: 16, weight: .light)
caption: .system(size: 14, weight: .light)
```

## 🌙 深色模式

默认就是深色主题！无需适配。

但需要确保：
- 文字对比度足够
- 图标清晰可见
- 渐变不过于刺眼
