# 🧘 梦境冥想与睡眠音效 - Phase 8 功能文档

> 帮助用户通过声音和冥想改善睡眠质量，增强梦境回忆能力

---

## 📋 功能概述

Phase 8 引入了完整的冥想与睡眠音效系统，包括：

1. **音效库** - 12 种精心挑选的助眠音效
2. **引导冥想** - 5 种专业引导音频脚本
3. **混音器** - 自定义音效组合
4. **我的预设** - 保存和加载个人配置

## 🎵 音效库

### 自然类 (Nature)

| 音效 | 描述 | 适用场景 |
|------|------|----------|
| 🌧️ 雨声 | 舒缓的雨声，帮助放松入睡 | 入睡困难、需要白噪音屏蔽 |
| 🌊 海浪 | 海浪拍打沙滩，带来宁静感 | 放松冥想、减压 |
| 🌲 森林 | 森林中的自然声音，鸟鸣与微风 | 晨间冥想、自然连接 |

### 噪音类 (Noise)

| 音效 | 描述 | 适用场景 |
|------|------|----------|
| ⚪ 白噪音 | 均匀的背景音，屏蔽外界噪音 | 专注工作、入睡 |
| 🌸 粉红噪音 | 更柔和的噪音，适合深度睡眠 | 深度睡眠、学习 |
| 🟤 棕色噪音 | 低沉的噪音，最适合专注 | ADHD 专注、深度放松 |

### 双耳节拍类 (Binaural Beats)

双耳节拍通过向左右耳播放略有差异的频率，在大脑中产生第三频率，影响脑波状态。

| 音效 | 频率 | 脑波 | 效果 |
|------|------|------|------|
| 🧠 40Hz | γ波 | 高频 | 提升意识、专注力、认知处理 |
| 😌 10Hz | α波 | 中频 | 放松、冥想、创意状态 |
| 🌙 5Hz | θ波 | 低频 | 深度放松、REM 睡眠、潜意识访问 |
| 💤 1Hz | δ波 | 超低频 | 深度无梦睡眠、身体修复 |

### 冥想类 (Meditation)

| 音效 | 描述 | 适用场景 |
|------|------|----------|
| 🔔 颂钵 | 藏式颂钵，帮助进入冥想状态 | 冥想开始/结束、脉轮平衡 |
| 🎐 风铃 | 轻柔的风铃声，带来平静 | 背景氛围、正念练习 |

## 🧘 引导冥想

### 1. 增强梦境回忆 (10 分钟)

**目标**: 提高醒来后的梦境记忆清晰度

**脚本流程**:
1. 放松身体，调整呼吸
2. 进入宁静空间想象
3. 设定"我会记住我的梦"意图
4. 重复强化意图
5. 锚定感觉，慢慢返回

**最佳使用时间**: 睡前或醒来后

---

### 2. 清醒梦诱导 (15 分钟)

**目标**: 帮助在梦中保持意识 (MILD 技巧)

**脚本流程**:
1. 全身渐进式放松
2. 想象正在做梦
3. 练习现实检查
4. 体验觉醒时刻
5. 锚定觉醒感觉

**最佳使用时间**: 睡前，配合 WBTB 技巧效果更佳

---

### 3. 睡前准备 (20 分钟)

**目标**: 渐进式放松，为高质量睡眠做准备

**脚本流程**:
1. 舒适姿势准备
2. 从脚趾到头顶的全身扫描
3. 感受每个部位放松
4. 呼吸放缓
5. 自然进入睡眠

**最佳使用时间**: 睡前 30 分钟

---

### 4. 减压放松 (10 分钟)

**目标**: 释放日常压力，让心灵平静

**脚本流程**:
1. 深呼吸三次
2. 想象压力为黑色烟雾
3. 呼气释放压力
4. 吸气吸入金色光
5. 感受平静

**最佳使用时间**: 任何需要放松的时刻

---

### 5. 晨间锚定 (5 分钟)

**目标**: 回顾梦境，锚定记忆

**脚本流程**:
1. 醒来后保持静止
2. 回想梦境画面和感觉
3. 给梦境命名
4. 记住关键情绪
5. 带着智慧开始一天

**最佳使用时间**: 醒来后，起床前

## 🎛️ 混音器功能

### 功能特性

- **多音效混合**: 同时播放多个音效
- **独立音量**: 每个音效单独控制音量 (0-100%)
- **总音量**: 全局音量控制
- **定时关闭**: 0/15/30/45/60/90 分钟选项
- **预设保存**: 保存自定义配置

### 默认预设

#### 🌧️ 雨夜好眠
- 雨声 (80%)
- 棕色噪音 (30%)
- 定时：30 分钟

#### 🌊 海边冥想
- 海浪 (70%)
- 风铃 (40%)
- 定时：20 分钟

#### 🧘 深度放松
- 5Hz 双耳节拍 (60%)
- 颂钵 (30%)
- 定时：15 分钟

#### 🌲 森林清晨
- 森林 (80%)
- 风铃 (50%)
- 定时：持续播放

## 🏗️ 技术实现

### 文件结构

```
DreamLog/
├── MeditationService.swift      # 核心服务
│   ├── SoundType               # 音效类型枚举
│   ├── SoundCategory           # 音效分类
│   ├── GuidedMeditationType    # 引导冥想类型
│   ├── SoundMix                # 混音配置结构
│   └── MeditationService       # 播放管理服务
│
└── MeditationView.swift         # UI 界面
    ├── MeditationView          # 主容器 (TabView)
    ├── SoundLibraryView        # 音效库
    ├── GuidedMeditationListView # 引导冥想列表
    ├── SoundMixerView          # 混音器
    ├── MyPresetsView           # 我的预设
    └── MiniPlayerView          # 迷你播放器
```

### 关键代码

#### 播放混音

```swift
let mix = SoundMix(
    name: "自定义混音",
    sounds: [
        SoundMix.SoundMixItem(soundType: .rain, volume: 0.8, isEnabled: true),
        SoundMix.SoundMixItem(soundType: .brownNoise, volume: 0.3, isEnabled: true)
    ],
    masterVolume: 1.0,
    timer: 30 * 60  // 30 分钟
)

MeditationService.shared.playMix(mix)
```

#### 播放引导冥想

```swift
MeditationService.shared.playGuidedMeditation(.dreamRecall)
```

## 📱 使用场景

### 场景 1: 入睡困难

**推荐配置**:
- 音效：雨声 + 棕色噪音
- 音量：60%
- 定时：30 分钟

### 场景 2: 想要记住梦境

**推荐配置**:
- 引导冥想：增强梦境回忆
- 时间：睡前或醒来后

### 场景 3: 尝试清醒梦

**推荐配置**:
- 引导冥想：清醒梦诱导
- 音效：5Hz 双耳节拍 (背景)
- 时间：睡前，配合 WBTB

### 场景 4: 午后小憩

**推荐配置**:
- 音效：森林 + 风铃
- 音量：40%
- 定时：20 分钟

### 场景 5: 专注工作

**推荐配置**:
- 音效：棕色噪音 或 40Hz 双耳节拍
- 音量：50%
- 定时：60 分钟
