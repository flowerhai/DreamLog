## 🔧 技术实现

### 架构设计

```
┌─────────────────────────────────────────────────────────┐
│                   DreamEBookExportView                   │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────────┐   │
│  │ 基本信息     │ │ 主题选择     │ │ 日期范围         │   │
│  └─────────────┘ └─────────────┘ └─────────────────┘   │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────────┐   │
│  │ 章节配置     │ │ 导出选项     │ │ 预览统计         │   │
│  └─────────────┘ └─────────────┘ └─────────────────┘   │
└─────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────┐
│              DreamEBookExportViewModel                   │
│  - config: EBookExportConfig                             │
│  - isExporting: Bool                                     │
│  - currentProgress/totalProgress: Int                    │
└─────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────┐
│               DreamEBookExportService                    │
│  - fetchDreams()                                         │
│  - organizeChapters()                                    │
│  - generatePDF()                                         │
│  - saveEBook()                                           │
└─────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────┐
│              DreamPDFExportRenderer                      │
│  (现有组件复用)                                          │
└─────────────────────────────────────────────────────────┘
```

### 数据流

1. **用户配置** → `EBookExportConfig`
2. **获取梦境** → `fetchDreams(for:includeIds:)`
3. **组织章节** → `organizeChapters(from:config:)`
4. **生成 PDF** → `generatePDF(chapters:config:)`
5. **保存文件** → `saveEBook(data:config:)`
6. **返回结果** → `.success(URL)` / `.failure(String)`

### 状态管理

```swift
enum EBookExportStatus {
    case idle
    case preparing
    case generating(Int, Int)  // 当前进度，总进度
    case completing
    case success(URL)
    case failure(String)
}
```

## 🎯 使用场景

### 场景 1: 月度梦境集

1. 选择「月度精选」预设模板
2. 日期范围选择「最近 30 天」
3. 选择「梦幻紫色」主题
4. 点击导出 → 生成月度梦境电子书

### 场景 2: 年度回顾

1. 选择「年度回顾」预设模板
2. 日期范围选择「今年」或「去年」
3. 选择「奢华金色」主题
4. 勾选「包含 AI 分析」
5. 点击导出 → 生成年度梦境精华

### 场景 3: 情绪主题集

1. 自定义章节，按情绪分类
2. 添加章节：「快乐梦境」「焦虑梦境」「平静梦境」
3. 为每个章节手动选择梦境
4. 选择「极简主义」主题
5. 点击导出 → 生成情绪主题电子书
