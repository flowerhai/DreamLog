## 🛠️ 制作方法

### 方法 1: Xcode Simulator (推荐)

**步骤**:

1. **启动 Simulator**
   ```bash
   open -a Simulator
   ```

2. **选择设备**
   - File → Open Simulator → iPhone 16 Pro Max (6.7")
   - 或 iPhone 16 Pro (6.1")

3. **运行应用**
   ```bash
   cd /path/to/DreamLog
   xcodebuild -scheme DreamLog -destination 'platform=iOS Simulator,name=iPhone 16 Pro Max' run
   ```

4. **导航到目标页面**
   - 手动操作或运行自动化脚本

5. **截取屏幕**
   ```bash
   xcrun simctl io booted screenshot ~/Desktop/screenshot1.png
   ```

6. **调整尺寸** (如需要)
   - 使用 Preview 或 Photoshop
   - 确保符合目标分辨率

### 方法 2: 真机截图

**步骤**:

1. **连接 iPhone 到 Mac**
2. **打开 Xcode → Window → Devices and Simulators**
3. **选择设备 → View Device Logs**
4. **手动在手机上截图**
5. **通过 AirDrop 或数据线传输到 Mac**

### 方法 3: Fastlane Snapshot (自动化)

**安装**:
```bash
sudo gem install snapshot
```

**配置** (`Snapshotfile`):
```ruby
scheme "DreamLog"
devices [
  "iPhone 16 Pro Max",
  "iPhone 16 Pro"
]
output_directory "./screenshots"
skip_open_summary true
```

**运行**:
```bash
snapshot
```

## 🎨 文案叠加设计

### 使用工具
- **Photoshop** - 专业级编辑
- **Figma** - 免费，适合 UI 设计师
- **Canva** - 在线工具，模板丰富
- **Preview** (macOS) - 简单标注

### 设计规范

**字体**:
- 中文：苹方 (PingFang SC)
- 英文：SF Pro Display
- 字重：标题 Bold / 副标题 Regular

**颜色**:
- 主文字：#FFFFFF (白色)
- 阴影：#000000 30% 不透明度
- 强调色：#9B7EBD (DreamLog 紫)

**位置**:
- 标题：顶部 15% 处，居中
- 副标题：标题下方 20pt
- 留白：左右各 40pt

**效果**:
- 文字阴影：X:0, Y:2, Blur:4
- 渐变遮罩：底部向上 30% 渐变透明

## 📤 导出设置

### Photoshop 导出

1. **文件 → 导出 → 存储为 Web 所用格式**
2. **格式**: PNG-24
3. **颜色**: sRGB
4. **优化**: 勾选"交错"
5. **文件大小**: 单张 < 10MB

### Figma 导出

1. **选中 Frame**
2. **右侧面板 → Export**
3. **格式**: PNG
4. **倍率**: 1x (已是目标分辨率)
5. **导出**

## 📞 问题排查

### 常见问题

**Q: 截图尺寸不对？**
A: 使用 Xcode Simulator 或调整画布尺寸到精确像素。

**Q: 文案在截图上不清晰？**
A: 增加文字阴影或使用渐变遮罩提高对比度。

**Q: 截图文件太大？**
A: 使用 PNG-8 或适当降低 JPEG 质量 (85%+)。

**Q: 状态栏不统一？**
A: 使用工具统一添加或移除状态栏。
