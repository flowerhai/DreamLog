## 📋 Phase 66 概述

Phase 66 将增强 AI 梦境解析功能，提供更深度、更个性化的梦境解读和洞察。基于现有的 AI 分析基础设施，本阶段将实现多层级梦境符号解读、跨梦境模式识别、趋势预测和个性化洞察生成。

### 核心目标

- 提供专业级的梦境符号解读（表面/心理/精神三层级）
- 实现跨梦境的模式识别和关联发现
- 基于历史数据预测梦境趋势
- 生成个性化的行动建议
- 构建可扩展的梦境符号知识库

---

## 🎯 核心功能

### 1. 🧠 深度梦境解析

**三层级解读**:
- **表面层** - 梦境内容的直接描述和字面含义
- **心理层** - 心理学视角的潜意识解读（弗洛伊德/荣格理论）
- **精神层** - 精神成长和自我发现的启示

**解读维度**:
- 梦境符号解析（人物/地点/物体/动作）
- 情绪模式分析
- 主题识别（追逐/坠落/飞行/考试/裸露等）
- 颜色象征意义
- 数字象征意义

### 2. 🔗 梦境关联分析

**关联类型**:
- 共同标签关联
- 共同情绪关联
- 相似内容关联（语义相似度）
- 时间接近关联
- 相似主题关联
- 清醒梦关联

**模式识别**:
- 重复出现的符号
- 周期性梦境主题
- 情绪变化趋势
- 梦境强度波动
- 清醒梦频率变化

### 3. 📈 趋势预测

**预测维度**:
- 未来 7 天梦境主题预测
- 情绪趋势预测
- 清醒梦概率预测
- 梦境清晰度趋势
- 潜在压力源预警

**算法**:
- 时间序列分析
- 周期性模式检测
- 外部因素关联（睡眠质量/压力水平/生活事件）

### 4. 🎯 个性化洞察

**用户画像维度**:
- 记录习惯（频率/时间/详细程度）
- 梦境特征（清晰度/强度/情绪分布）
- 兴趣偏好（标签/主题）
- 成长轨迹（清醒梦进展/情绪变化）

**洞察类型**:
- 记录习惯建议
- 清醒梦训练建议
- 情绪管理建议
- 创意启发建议
- 睡眠质量改善建议

### 5. 💡 行动建议

**建议类别**:
- 📝 记录优化（如何记录更有效的梦境）
- 🧘 冥想练习（针对性的冥想主题）
- 🎨 创意表达（艺术/写作/音乐）
- 💤 睡眠改善（睡前仪式/环境优化）
- 🧠 自我探索（反思问题/日记提示）
- 🌱 个人成长（基于梦境启示的行动）

### 6. 📚 梦境词典

**符号分类**:
- 👤 人物（家人/朋友/陌生人/名人/已故者）
- 🏠 地点（家/学校/工作/陌生地方/奇幻场景）
- 🎯 物体（日常用品/交通工具/动物/自然元素）
- 🏃 动作（追逐/逃跑/飞行/坠落/战斗/性）
- 🎭 情境（考试/迟到/迷路/被追赶/公开演讲）

**文化背景**:
- 西方心理学解读
- 中国传统文化解读
- 印度文化解读
- 原住民文化解读

## 🔧 技术实现

### 架构模式

- **MVVM** - 模型 - 视图 - 视图模型
- **Actor** - 异步并发安全
- **SwiftData** - 数据持久化
- **Protocol-oriented** - 协议导向设计

### 关键技术

- **NaturalLanguage** - 文本分析和语义理解
- **CoreML** - 机器学习模型（可选）
- **SwiftUI** - 声明式 UI
- **Combine** - 响应式数据流

### 数据流

```
DreamEntry → DreamAIAnalysisService → DreamSymbolDictionary
                                    → DreamPatternRecognition
                                    → DreamInsightGenerator
                                    → DreamAnalysis (结果)
                                    → UI 展示
```
