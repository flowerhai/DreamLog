# DreamLog 🌙 - AI 梦境日记

> 记录你的梦，发现潜意识的秘密

---

## 💡 创意来源

每个人都会做梦，但醒来就忘了。DreamLog 帮你记录梦境，用 AI 分析其中的模式和隐藏含义。

---

## 🎯 核心功能

### 1. 快速记录
- 醒来后语音输入梦境
- AI 自动转文字并整理
- 支持添加标签（人物、地点、情绪）

### 2. AI 梦境解析
- 分析梦境中的象征意义
- 关联近期生活事件
- 生成"梦境报告"

### 3. 梦境模式发现
- "你最近经常梦到水"
- "压力大时容易梦到追逐"
- "周五晚上的梦更奇幻"

### 4. 梦境画廊
- 用 AI 绘画把梦境可视化
- 创建你的"梦境博物馆"
- 分享有趣的梦（匿名）

### 5. 清醒梦训练
- 提供清醒梦技巧
- 记录清醒梦体验
- 追踪训练进度

## 🧠 梦境知识库

### 常见梦境象征
| 梦境元素 | 可能含义 |
|---------|---------|
| 掉落 | 失控感、焦虑 |
| 飞行 | 自由、解脱 |
| 追逐 | 逃避问题 |
| 牙齿掉落 | 变化、成长 |
| 水 | 情绪、潜意识 |
| 考试 | 被评判的焦虑 |

## 📋 技术栈

- **前端**: SwiftUI (iOS 16+)
- **语音**: Speech Framework
- **AI 分析**: LLM API
- **AI 绘画**: Stable Diffusion API
- **存储**: SwiftData + iCloud

## 🔒 隐私保护

- **本地优先** - 梦境数据默认本地存储
- **端到端加密** - iCloud 同步时加密
- **匿名分享** - 分享时自动移除个人信息
- **可删除** - 一键删除所有数据
