### 修复 2: 重命名 LoadingView (SleepDataView.swift)

**问题**: `LoadingView` 在多个文件中定义，导致编译错误。

**修复前**:
```swift
// DreamLog/CommonViews.swift:55
struct LoadingView: View { ... }

// DreamLog/SleepDataView.swift:191
struct LoadingView: View { ... }  // ❌ 重复
```

**修复后**:
```swift
// DreamLog/CommonViews.swift:55
struct LoadingView: View { ... }  // ✅ 保留通用版本

// DreamLog/SleepDataView.swift:191
struct SleepDataLoadingView: View { ... }  // ✅ 重命名为特定版本
```

**更新引用**:
```swift
// SleepDataView.swift:26
LoadingView()  →  SleepDataLoadingView()
```
