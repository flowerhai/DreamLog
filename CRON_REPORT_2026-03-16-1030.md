### 2. 优化 PDF 导出标题生成 🔧

**文件**: `DreamExportHubService.swift`  
**变更**: 1 行

**问题**: 使用强制解包 `options.template!` 虽然安全但不符合 Swift 最佳实践。

**修复**: 使用 `map` 优雅处理可选值。

```diff
- title: options.template != nil ? "梦境导出 - \(options.template!)" : "梦境记录"
+ title: options.template.map { "梦境导出 - \($0)" } ?? "梦境记录"
```

**提交**: `37fd1d6 refactor(export): 优化 PDF 导出标题生成 - 使用 map 替代强制解包 🔧✨`
