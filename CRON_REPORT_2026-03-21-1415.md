## 🔧 代码质量改进

### 修复：预览模型容器创建

**问题**: 多个 View 的 Preview 中使用强制解包创建 ModelContainer

**修复文件**:
- `DreamMorningReflectionView.swift`
- `DreamSmartInsightsView.swift`

**修复内容**:
```swift
// 修复前
let container = (try? ModelContainer(...)) ?? (try! ModelContainer(...))

// 修复后
do {
    container = try ModelContainer(...)
} catch {
    fatalError("Failed to create preview model container: \(error)")
}
```

**影响**: 更清晰的错误信息，便于调试
