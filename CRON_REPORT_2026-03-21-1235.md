## 🔧 代码质量改进

### 修复：DreamPatternPredictionService 强制解包

**问题**: Calendar 日期计算使用 `!` 强制解包，可能导致运行时崩溃

**文件**: `DreamLog/DreamPatternPredictionService.swift`

**修复内容**:
```swift
// 修复前
let olderDreams = dreams.filter { $0.createdAt >= calendar.date(byAdding: .day, value: -days * 2, to: Date())! }

// 修复后
let olderCutoffDate = calendar.date(byAdding: .day, value: -days * 2, to: Date()) ?? Date()
let olderDreams = dreams.filter { $0.createdAt >= olderCutoffDate }
```

**影响**: 避免在极端情况下（如日历计算失败）导致应用崩溃
