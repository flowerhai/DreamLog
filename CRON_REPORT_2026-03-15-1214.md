## 🛠️ 技术亮点

### 1. 模板变量系统

```swift
// 支持的变量
{{title}}          // 梦境标题
{{content}}        // 梦境内容
{{date}}           // 记录日期
{{tags}}           // 标签列表
{{tagsJoined}}     // 带#标签 (#飞行 #自由)
{{emotions}}       // 情绪列表
{{aiAnalysis}}     // AI 解析
{{contentTruncated}} // 截断内容 (适配 Twitter)
```

### 2. 条件语句支持

```handlebars
{{#if aiAnalysis}}
## AI 梦境解析

{{aiAnalysis}}
{{/if}}

{{#if tags}}
标签：{{tags}}
{{/if}}
```

### 3. 预设模板

- **Medium 风格** - 长文格式，适合深度内容
- **微信公众号风格** - 中文排版，emoji 装饰
- **小红书风格** - 短小精悍，标签丰富
- **Twitter/X 风格** - 280 字符限制，自动截断
- **邮件通讯风格** - HTML 格式，支持多篇梦境
